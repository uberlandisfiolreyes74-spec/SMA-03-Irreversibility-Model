## 🧮 Calculadora interactiva SMA-03 Lite

Pon a prueba el modelo con tus propios valores y obtén una predicción instantánea del **Punto de No Retorno (PNR)**.  
La calculadora implementa los coeficientes reales de la regresión logística validada con 113 casos (80% de precisión).

👉 **Accede a la calculadora:**  
- Si tienes el repositorio clonado, abre `calculator/index.html` en tu navegador.  
- O pruébala online (si activaste GitHub Pages) en:  
  [`https://uberlandsifoldreyes74.github.io/SMA-03-Irreversibility-Model/calculator/`](https://uberlandsifoldreyes74.github.io/SMA-03-Irreversibility-Model/calculator/)

**Características:**
- Interfaz intuitiva con sliders para todas las variables del modelo.
- Cálculo en tiempo real del PNR y nivel de riesgo.
- Informe PDF personalizado (disponible tras simular un pago de prueba).
- Código 100% abierto y modificable.

> **Nota:** El botón de pago es solo una demostración; puedes configurar tu propio sistema de monetización si lo deseas.
